# 项目文档索引

欢迎来到本项目文档！

本项目从一个简单的准分布式水文模型开始，逐步扩展为一个包含多种水文算法、支持GIS自动流域划分、并具备EnKF数据同化能力的综合性水文模拟与分析框架。

## 文档结构

为了更好地理解项目的各个部分，请按顺序阅读以下文档：

1.  **[基础水文模型示例](./basic_model_example.md)**
    - 介绍了项目最初的、基于分区的准分布式水文模型的结构和用法。

2.  **[GIS流域划分与参数分区示例](./gis_delineation_example.md)**
    - 介绍了如何使用`whitebox-tools`和`geopandas`，根据DEM等基础数据自动划分亚流域，并结合土地利用和土壤类型数据，为每个亚流域创建参数分区ID。

3.  **[EnKF参数率定与数据同化示例](./enkf_calibration_example.md)**
    - 介绍了如何使用集合卡尔曼滤波（EnKF）和增广状态向量技术，对水文模型进行参数率定和状态更新，以提高模拟精度。

4.  **[深度学习模型](./dl_models.md)**
    - 介绍了如何使用框架中集成的LSTM和GNN模型进行流量预测。

5.  **[单个水文算法模块演示](./algorithm_examples.md)**
    - 包含了对项目中集成的各个独立的产汇流算法模块（如SCS、马斯京根法、单位线法、马斯京根-康基法）的简单功能演示和说明。
