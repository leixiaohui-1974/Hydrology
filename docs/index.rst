Overview
--------

The Hydrology Framework is a modular, extensible platform for hydrological modeling that supports:

* 1D hydraulic modeling using the Preissmann scheme
* 2D hydraulic modeling with finite volume methods
* Deep learning integration (LSTM and GNN models)
* Real-time data processing and forecasting
* Performance monitoring and optimization
* Security and audit logging
* Web-based user interface

Key Features
------------

* **Modular Architecture**: Component-based design for easy extension
* **Multiple Model Types**: Support for 1D, 2D, and ML-based models
* **Real-time Processing**: Live data ingestion and processing capabilities
* **Performance Monitoring**: Built-in performance tracking and optimization
* **Security**: Comprehensive security and audit logging
* **Web Interface**: Modern web-based GUI for model configuration and visualization
* **API Integration**: RESTful API for external system integration
