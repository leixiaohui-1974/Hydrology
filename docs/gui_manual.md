# 快速建模工具 - GUI手册

## 1. 简介

欢迎使用快速水模型建模工具！本工具提供一个图形用户界面（GUI），用以可视化地构建、运行并分析复杂的耦合水系统模型。

您无需编写代码或手动创建配置文件，即可通过本界面拖放组件、将它们连接成网络，并以实时反馈的方式运行仿真。

## 2. 开始使用

要启动图形界面，请在项目的根目录中运行以下命令：

```bash
python3 gui/main.py
```

这将打开主应用程序窗口。

## 3. 界面概览

GUI分为三个主要面板，底部还有一个结果显示区。

### 3.1. 左侧：组件面板

此面板包含模型中所有可用的构建模块。

-   **🏞️ 流域 (Catchment):** 根据降雨产生径流的水文组件。
-   **🌊 河道 (River Reach):** 用于河段的一维水动力学模型。
-   **➕ 交叉点 (Junction):** 用于合并或分流的节点。
-   **⛕ 闸门 (Gate):** 用于控制河道水流的闸门结构。
-   **ポンプ 水泵 (Pump):** 为水流增加能量（水头）的水泵。

**使用方法:** 从此面板中单击并拖动任何组件，然后将其放置到中心的网络画布上。

### 3.2. 中心：网络画布

这是您的主要工作区。

-   **放置节点:** 从组件面板拖动组件以节点的形式放置在画布上。每个节点都会被赋予一个默认的唯一名称（例如 `RiverReach_1`）。
-   **连接节点:** 要连接两个节点（例如，从一个流域到一个河道），请使用两点点击法：
    1.  在**源**节点（例如流域）上单击一次。它将被红色边框高亮显示。
    2.  在**目标**节点（例如河道）上再次单击。一条代表水流方向的箭头将被画出。

### 3.3. 右侧：属性面板

此面板是上下文相关的，允许您编辑模型的参数。

-   **选择节点:** 在画布上单击任何节点。它将被绿色高亮显示，其参数将出现在属性面板中。
-   **编辑参数:** 您可以更改输入字段中的任何值（例如 `slope`、`manning_n`、`CN`）。更改会自动保存。组件的名称也可以在此处更改。
-   **全局设置:** 未选择任何节点时，此面板将显示全局仿真设置和仿真结束后的绘图控件。

## 4. 配置数据源和预处理

在左侧面板的“组件”部分下方，新增了两个面板，用于配置高级数据输入和预处理步骤。

### 4.1. 数据源 (Areal Precipitation)

此面板允许您配置降雨输入。默认情况下，模型使用一个集总的降雨序列。但是，您可以配置一个基于多个站点的面雨量插值。

[Screenshot of Data Sources panel]

1.  **选择降雨输入类型:**
    -   `Lumped Rainfall Series`: 默认选项，使用单一时间序列。
    -   `Interpolated Areal Rainfall`: 启用面雨量插值。
2.  **设置插值选项** (当选择 `Interpolated` 时显示):
    -   **Subbasins Shapefile:** 选择定义子流域边界的 `.shp` 文件。
    -   **Rain Gauges CSV:** 选择定义雨量站位置和ID的 `.csv` 文件。
    -   **Method:** 从下拉菜单中选择插值方法（IDW, Thiessen, Kriging）。
    -   **方法参数:** 根据所选方法，下方会显示相应的参数输入框（例如，IDW的`power`，Thiessen的`cache_file`等）。

### 4.2. 预处理 (Preprocessing)

此面板允许您在主仿真运行之前，对数据进行验证和分析。目前支持基流分割预览。

[Screenshot of Preprocessing panel]

1.  **选择流量数据:** 从下拉菜单中选择要进行基流分割的流量数据源（此功能将在后续版本中动态填充）。
2.  **设置Alpha参数:** 输入用于Lyne-Hollick数字滤波器的 `alpha` 值。
3.  **点击 "Run & Preview Steps"**:
    -   后端将运行基流分割算法。
    -   完成后，一张包含总流量和分割出的基流的预览图将显示在下方的图片框中。

## 6. 完整工作流程：从零到运行

现在，GUI已成为一个功能齐全的建模工具。以下是一个完整的工作流程：

1.  **启动GUI**: `python3 gui/main.py`
2.  **配置数据源**:
    -   在左侧的“数据源”面板中，选择您的降雨输入类型。
    -   如果选择插值，请提供所需的文件和参数。
3.  **配置预处理步骤**:
    -   在“预处理”面板中，配置您希望在仿真前运行的任何步骤（例如，基流分割）。
    -   使用“预览”按钮来检查这些步骤的结果。
4.  **构建网络**:
    -   从组件面板拖放节点到画布上。
    -   通过两点点击法连接它们。
5.  **设置参数**:
    -   单击每个节点，并在右侧的属性面板中设置其物理参数。
    -   **重要**: 对于需要输入的组件（例如，水文模型需要降雨），请确保其**名称**与您在`global_inputs`中定义的数据源名称相匹配，或者使用高级配置中的数据映射功能。
6.  **运行仿真**:
    -   单击左上角的 **"Run" 按钮**。
    -   程序现在将 **动态地** 根据您在GUI中的所有设置（节点、连接、参数、数据源配置）来构建和运行模型。它不再运行一个硬编码的示例。

### 实时反馈

仿真运行时，您可以在窗口底部监控其进度：

-   **图表（左侧）:** 显示网络中最终组件出流量的实时绘图。
-   **日志（右侧）:** 显示仿真的分步进度日志，包括所有预处理步骤的输出。

## 7. 分析和保存

### 分析结果
仿真成功结束后：

1.  **绘图控件**将出现在右上方的面板中。
2.  **选择组件:** 使用第一个下拉菜单选择您想检查的组件。
3.  **选择变量:** 第二个下拉菜单将自动填充该组件可用的变量。
4.  **点击 "Plot Selected"**: 底部的图表将更新，显示所选数据的完整时间序列。

### 保存您的工作
1.  在GUI中构建完网络并设置好所有参数后，点击 **"Save"（保存）按钮**。

仿真成功结束后：

1.  **绘图控件**将出现在右上方的面板中。
2.  **选择组件:** 使用第一个下拉菜单选择您想检查的组件（例如 `R1_main_river`）。
3.  **选择变量:** 第二个下拉菜单将自动填充该组件可用的变量（例如 `Q`, `Z`, `outflow`）。
4.  **点击 "Plot Selected"（绘制所选）:** 底部的图表将更新，显示所选数据的完整时间序列。对于具有多个节点的变量（如河道中的`Q`和`Z`），它将绘制最后一个节点的数据。

## 8. 保存您的工作

1.  在GUI中构建完网络并设置好所有参数后，点击 **"Save"（保存）按钮**。
2.  这将调用Python后端，将您的可视化模型转换为YAML配置格式。
3.  一个名为 `gui_output_config.yaml` 的文件将被保存在项目的根目录中。该文件之后可以被 `run_from_config.py` 脚本使用。
