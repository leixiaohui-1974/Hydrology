# 第11章：水文分析中的GIS数据

到目前为止，我们都是在“概念”层面定义我们的流域和河网（例如，通过手动创建`catchment_definition.csv`或在代码中添加节点和河段）。但在实际应用中，水文模型的构建离不开真实的地理空间信息。

本章将介绍在我们的框架中，用于自动化流域分析的几种关键GIS数据。

## 什么是GIS数据？

GIS（地理信息系统）数据是包含地理位置信息的数据。它们通常分为两大类：

1.  **栅格数据 (Raster Data)**: 由像素（或单元格）组成的网格，每个像素都有一个值。它非常适合表示连续变化的地理现象，如地形高程、温度、降雨分布等。
    -   **文件格式**: `.tif` (GeoTIFF), `.asc`, `.img`, 等。

2.  **矢量数据 (Vector Data)**: 由点、线、多边形这些离散的几何对象组成。它非常适合表示具有明确边界的地理实体。
    -   **点 (Point)**: 水文站、雨量站、水库。
    -   **线 (Line)**: 河流、道路。
    -   **多边形 (Polygon)**: 湖泊、行政区、土地利用类型、土壤类型。
    -   **文件格式**: `.shp` (ESRI Shapefile), `.geojson`, `.kml`, 等。

## 本框架使用的关键GIS数据

在我们的GIS流域划分工具中，主要使用了以下几种数据：

### 1. 数字高程模型 (DEM)

-   **文件**: `gis_data/dem.tif`
-   **类型**: 栅格数据
-   **作用**: **这是所有水文分析的基础**。DEM的每一个像素值都代表了该点的地面高程。通过分析DEM，我们可以推断出水往哪里流，从而确定流向、计算汇流累积量、提取河网、并最终划分出流域和子流域的边界。

### 2. 土地利用数据

-   **文件**: `gis_data/land_use.shp`
-   **类型**: 矢量数据（多边形）
-   **作用**: 描述了地表的覆盖类型，如森林、草地、城市、水体等。土地利用类型直接影响着流域的产汇流特性。例如：
    -   **产流**: 城市不透水地面的产流能力远高于森林。在SCS模型中，不同的土地利用类型对应着不同的**CN值**。
    -   **汇流**: 植被茂密的区域，其地表糙率更大，会减慢水流的速度。

### 3. 土壤类型数据

-   **文件**: `gis_data/soil.shp`
-   **类型**: 矢量数据（多边形）
-   **作用**: 描述了地表土壤的物理属性，如砂土、黏土、壤土等。土壤是影响产流和入渗的关键因素。
    -   **产流**: 砂土的渗透性好，产流能力弱；而黏土的渗透性差，产流能力强。这同样会影响到模型的**CN值**或**土壤蓄水容量**等参数。

## 数据的来源

在实际研究中，这些数据可以从多种渠道获取：
-   **DEM**: 可以从全球或区域性的公开数据集中下载，如SRTM (Shuttle Radar Topography Mission)、ASTER GDEM，或各国测绘部门发布的高精度数据。
-   **土地利用/土壤**: 通常可以从各国的土地、农业、环境等部门的地理信息平台获取。

## 总结

DEM、土地利用和土壤类型是进行精细化水文模拟的三大基础下垫面数据。理解它们的作用，是进行自动化流域划分和参数分区的前提。
