# 第8章：高级汇流：马斯京根-康基法

在上一章中，我们介绍了经典的马斯京根法，它需要两个经验性参数 `K` 和 `x`。这种方法虽然简单，但参数的物理意义不明确，需要通过历史数据率定。

本章，我们将介绍一个更高级的“降阶水动力模型”——**马斯京根-康基法 (Muskingum-Cunge)**。

## 什么是马斯京根-康基法？

它是一种水力学汇流法，巧妙地将马斯京根法的简洁性与水动力学方程的物理基础结合了起来。

其核心思想是：**参数 `K` 和 `x` 不再是需要率定的经验常数，而是可以根据河道的物理特性和当前的水流条件，在每一个时间步动态计算得出。**

这使得模型能够更真实地反映洪水波在河道中的传播和变形规律。

## 核心逻辑

在 `hydro_model/routing.py` 的 `MuskingumCungeRouting` 模块中，我们实现了这个逻辑。在每个时间步的 `run` 方法内部，程序会执行以下计算：

1.  **估算平均流量 `Q_avg`**: 根据当前和前一时间步的入流，估算出河段内的平均流量。
2.  **计算水力学要素**: 利用**曼宁公式**，根据平均流量和河道断面参数（宽度、坡度、糙率），反算出河段的平均**水深 `y`** 和**流速 `v`**。
3.  **计算波速 `c`**: 水流扰动在河道中传播的速度，即波速，可以通过流速 `v` 计算得出。
4.  **计算水力扩散系数 `q`**: 这是一个描述洪水波“展平”趋势的物理量。
5.  **动态计算 `K` 和 `x`**:
    -   `K = length / c` (蓄流时间常数等于河段长度除以波速)
    -   `x = 0.5 * (1 - q / (c * length))` (权重因子与水力扩散和波速有关)
6.  **应用马斯京根方程**: 使用这些动态计算出的 `K` 和 `x`，通过标准的马斯京根方程计算出当前的出流量 `O_t`。

## 主要参数

与简单的马斯京根法不同，此模块的参数都是具有明确物理意义的河道属性：
-   `length`: 河段长度 (m)。
-   `slope`: 河床比降 (m/m)。
-   `manning_n`: 曼宁糙率系数，代表河床的粗糙程度。
-   `width`: 河道宽度 (m)，在本实现中，我们假设为宽浅矩形河道。

## 示例

**脚本:** `examples/run_muskingum_cunge_example.py`

这个脚本构建了一个具有特定物理参数的河段，并演示了如何使用`MuskingumCungeRouting`模块。

```python
# examples/run_muskingum_cunge_example.py

# 1. 定义河道的物理参数
params = {
    'length': 5000,    # 5 km
    'slope': 0.0002,   # 比降
    'manning_n': 0.03, # 糙率
    'width': 50.0,     # 宽度
}

# 2. 创建汇流模块实例
mc_router = MuskingumCungeRouting(**params)

# 3. 输入一个洪水波并进行演算
# ...
```

运行该脚本，您可以观察到，即使我们没有提供任何经验性的 `K` 和 `x`，模型也能够成功地对洪水波进行演算。

马斯京根-康基法是一个非常强大的工具，它在水文学的简洁性和水力学的物理基础之间取得了完美的平衡。
