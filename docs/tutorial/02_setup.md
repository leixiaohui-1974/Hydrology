# 第2章：环境搭建

在开始探索这个强大的水文模拟框架之前，我们需要先确保您的计算机上已经安装了所有必要的工具。本章将指导您完成整个环境的搭建过程。

## 1. Python环境

本框架是基于 **Python 3.10** 或更高版本开发的。如果您尚未安装Python，请从官方网站 [python.org](https://www.python.org/) 下载并安装最新版本。

在安装时，请确保勾选了“Add Python to PATH”或类似选项，这样您就可以在命令行中直接使用`python`和`pip`命令。

## 2. 依赖库安装

本项目依赖于一系列强大的第三方Python库来进行科学计算、地理空间分析和数据可视化。您可以使用Python的包管理工具`pip`来一键安装它们。

打开您的命令行工具（如Windows的CMD或PowerShell，macOS或Linux的Terminal），然后运行以下命令：

```bash
pip install pandas matplotlib geopandas rasterio whitebox numpy
```

这个命令会告诉`pip`去下载并安装所有我们需要的库。

### 依赖库简介

让我们简单了解一下每个库的作用：

-   **`numpy`**: Python科学计算的基础。它提供了强大的多维数组对象和相关的数学函数，是我们所有数值计算的核心。
-   **`pandas`**: 一个快速、强大、灵活且易于使用的数据分析和操作库。我们将用它来处理时间序列数据，如降雨和流量数据。
-   **`matplotlib`**: 一个综合性的数据可视化库。我们将用它来绘制各种图表，如流量过程线、参数收敛图等。
-   **`geopandas`**: 一个让在Python中处理地理空间矢量数据（如点、线、多边形）变得简单的库。我们将用它来处理流域、土地利用等shapefile数据。
-   **`rasterio`**: 一个用于读写栅格数据（如DEM）的强大库。
-   **`whitebox`**: `WhiteboxTools`地理空间分析引擎的Python接口。这是我们进行自动化流域划分的核心工具。在您第一次使用它时，它会自动从网络上下载并配置好所需的二进制可执行文件。

## 3. 获取项目代码

当然，您还需要本项目的完整代码。假设您已经通过某种方式（如`git clone`或下载zip包）获取了整个项目，并解压到了您的工作目录。

## 4. 验证安装

当所有库都安装完成后，您可以进行一个简单的验证。在您的命令行中，进入项目的根目录，然后运行我们最基础的示例脚本：

```bash
python examples/run_example.py
```

如果脚本成功运行，并在`examples/results`目录下生成了`comparison_plot.png`等结果文件，那么恭喜您，您的环境已经准备就绪！
